# Messenger

### Description
This is a messenger. It has private chats between two users and group chats among several users.

### Stack
* NodeJS
* Express
* TypeScript
* Socket.IO
* JWT
* MongoDB

### Launch
To clone repository:
```
git clone https://github.com/Erkebulan-Akhmediya/chat.git
```

To start the project:
```
npm run dev
```
